Agregar nueva información al grafo de conocimiento

¿Cómo funciona?

1. Se realiza la conexión a base con la finalidad de exportar la información a formato csv.

2. Con el conjunto de dataset exportados se crea el grafo de conocimiento seguiendo el esquema definido utilizando los vocablos SKOS, SCHEMA y FOAF

3. Finalmente, se importa el nuevo conjunto de datos en formato rdf a la base de datos semántica.

Llene el siguiente formulario con la información de conexión a la base de datos.

Dominio Puerto Nombre BD Usuario Fecha Act.
